## 2. 리스트

### 2.1 Ordered List - 순서있는 리스트

```
1. Java
2. Python
3. Javascript
```

#### 결과
1. Java
2. Python
3. Javascript

### 2.2 Unordered List - 순서없는 리스트

```
- Language
    - Python
    - Java
        - spring
    - Javascript
```

#### 결과
- Language
    - Python
    - Java
        - Spring
    - Javascript

### 2.3 Todo List - 체크 리스트

```
- [ ] Language
    - [x] Python
    - [ ] Java
        - [x] Spring
    - [ ] Javascript
```

#### 결과
- [ ] Language
    - [x] Python
    - [ ] Java
        - [x] Spring
    - [ ] Javascript

### 2.4 Description List - 설명 리스트

```
Language
: Python, Java, Javascript

CS
: Data Structure, Algorithm, Network
```

#### 결과
Language
: Python, Java, Javascript

CS
: Data Structure, Algorithm, Network
